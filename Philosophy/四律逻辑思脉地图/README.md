# 四律 · 逻辑思脉地图（科普勒斯顿全谱系）Atlas Logoi

把**同一律、矛盾律、排中律、充足理由律**当作可装可卸的「**选择公理**」——
以 **《科普勒斯顿哲学史》全套**（Copleston, *A History of Philosophy*, Vol. I–IX）覆盖的思想家为基础，
沿**前苏格拉底 → 当代**摊开 **101 位哲学家**的逻辑思脉与历史分叉。纯静态单文件，**可部署到 GitHub Pages**。

《逻各斯原本》app 的同源姊妹篇的全谱系版。交互形式不变（开关 + 地图 + 四象限 + 松绑史）。

## 这一版做了什么

### 1. 以科普勒斯顿全套为基础大幅补充（26 个时代带，101 人）
前苏格拉底自然哲学（泰勒斯/阿那克西曼德/毕达哥拉斯/赫拉克利特/巴门尼德/芝诺）· 多元论与原子论（恩培多克勒/阿那克萨戈拉/德谟克利特）· 智者与苏格拉底学派（普罗泰戈拉/高尔吉亚/苏格拉底/安提斯泰尼/麦加拉的欧几里得）· 柏拉图与亚里士多德 · 希腊化罗马（伊壁鸠鲁/斯多葛/怀疑派/西塞罗）· **新柏拉图主义（普罗提诺/波菲利/普罗克洛）** · 教父与早期中世纪（奥古斯丁/波爱修/伪狄奥尼修斯/爱留根纳）· **经院盛期·阿拉伯与犹太（阿维森纳/阿威罗伊/安萨里/迈蒙尼德/安瑟伦/阿伯拉尔）** · 经院盛期·拉丁（大阿尔伯特/波纳文图拉/阿奎那）· 经院后期·神秘主义（司各脱/奥卡姆/埃克哈特/**库萨的尼古拉**）· 文艺复兴与科学革命（布鲁诺/培根/伽利略/霍布斯）· 大陆理性论（笛卡尔/帕斯卡/马勒伯朗士/斯宾诺莎/莱布尼茨）· 英国经验论（洛克/贝克莱/休谟）· 启蒙运动（伏尔泰/卢梭/狄德罗/拉美特利/沃尔夫）· 康德 · 德国唯心论（费希特/谢林/施莱尔马赫/黑格尔）· 黑格尔之后（费尔巴哈/马克思/叔本华/克尔凯郭尔/尼采）· 19世纪英美（边沁/密尔/斯宾塞/布拉德雷/皮尔士/詹姆斯/杜威）· 法国哲学（孔德/柏格森/马塞尔/梅洛庞蒂）· 现象学存在主义（胡塞尔/雅斯贝尔斯/海德格尔/萨特/加缪）· 过程与实在论奠基（怀特海/摩尔）· 分析哲学逻辑奠基（弗雷格/罗素/维特根斯坦/卡尔纳普/蒯因）· 元逻辑（塔尔斯基/哥德尔）· 非经典逻辑分叉（布劳威尔/卢卡西维茨/扎德/达科斯塔·普里斯特）· 后结构解构（德里达/德勒兹/福柯）· 当代思辨实在论（梅亚苏/巴迪欧）。

> 其中可见许多「先声」式分叉：安萨里的偶因论 → 休谟；库萨的「对立之重合」→ 黑格尔；亚里士多德「明日海战」→ 卢卡西维茨/布劳威尔。

### 2. 立场标注改为三色 × 镶环
每个节点下方四颗「签名点」（同·矛·排·理），按你的要求配色：

| 立场 | 颜色 |
| --- | --- |
| 持守 | 🟢 绿色 |
| 极致 | 🟢 绿色（镶环） |
| 松动 | 🔴 红色 |
| 否弃 | 🔴 红色（镶环） |
| 限定 | 🟡 黄色 |
| 悬置 | 🟡 黄色（镶环） |
| 不涉 | ○ 空心（不变） |

即：**绿＝持守 / 红＝松开 / 黄＝限定**，**镶环**为同族中更强的一档。

## 怎么用

* **顶部四开关**：每条律设 **装上·守 / 中立 / 卸下·松或弃**。拨动后地图高亮*符合该配置的哲学家*，并把彼此的**承继（实线）/对反（虚线 ⊣）**连成分叉；多开关叠加 = 取交集。
* **点任一哲学家**：读其全貌（四律立场 + 关键动作 + 注 + 承继/对反），并自动把四开关设成它的签名。
* **右上「现代逻辑四象限」**：矛盾 × 排中的形式收成（经典 / 直觉主义 / 次协调·双面真理 / 相关逻辑 FDE），随开关实时点亮；点格子即反设这两个开关。
* **▶ 松绑史**：自巴门尼德「四律全开 + 思维=存在」的锁死机器，逐代下行 26 幕。
* 地图较长，可在左侧滚动浏览。

## 部署到 GitHub Pages
**网页上传**：新建仓库 → Add file → Upload files 拖入 `index.html` → Settings → Pages → Source 选 `Deploy from a branch`，分支 `main`、目录 `/ (root)` → 访问 `https://<用户名>.github.io/<仓库名>/`。

**命令行**：
```bash
cd logic-atlas-copleston
git init && git add index.html && git commit -m "四律 逻辑思脉地图 科普勒斯顿全谱系"
git branch -M main && git remote add origin https://github.com/<用户名>/<仓库名>.git
git push -u origin main
```
然后在 Settings → Pages 选 `main / root` 开启。
