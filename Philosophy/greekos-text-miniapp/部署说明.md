# 十六组核心概念二元对立 · 去看古希腊哲学（文字版）— 部署说明（保姆级）

> 这一版是按你给的 **Word 文字稿** 重做的**原生文字小程序**：手机竖屏阅读模式，矢量文字、可缩放系统字号、整包仅约 30KB（无图片）。
> 文件包：`greekos-text-miniapp.zip`（解压得文件夹）。

## 0. 和上一版（图片放映）的区别
- 上一版是把 PPT 每页当**图片**翻看；这一版把**文字内容**重排成原生阅读界面——文字清晰、随手机字号自适应、可上下滚动顺读，更适合手机。
- 设计为**两种进入方式 + 一个阅读器**：
  - **总表**：十六组对立的卡片网格（编号 + 希腊文 + 中文），点任一组进入；
  - **历史脉络**：按讲义实际历史顺序分成五个阶段 + 一篇总纲，点章节或某组进入；
  - **阅读器**：单组详读（编号徽章 + 希腊文 + 中文 + 正文段落/要点列表），底部「上一组 / 目录 / 下一组」可顺着读完整本，含「回到顶部」。
- 注意脉络里**第四阶段**（柏拉图—亚里士多德）刻意按讲义的讲解顺序排列（11&12 → 10 → 8 → 9），而总表仍按 1–16 编号；两种次序都能到达同一篇阅读内容。

```
greekos-text-miniapp/
├─ app.js / app.json / app.wxss

└─ pages/index/

   ├─ index.js     ← 目录/阅读/翻页逻辑

```

## 2. 最快看到效果（免注册）
1. 安装**微信开发者工具（稳定版）**：`https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html`
2. 解压本 zip 得文件夹。
3. 打开工具 → 手机扫码登录 → **「+ 导入项目」** → 目录选该文件夹 → AppID 点 **「测试号」** → 导入。模拟器即可阅读；右上 **「预览」** 扫码上真机。

## 3. 正式上线
1. **注册账号**：`mp.weixin.qq.com` → 立即注册 → 选「小程序」→ 邮箱激活 → 选主体（个人/企业）→ 完成验证 → 「开发→开发管理→开发设置」复制 **AppID**。
2. **填 AppID**：开发者工具「详情→基本信息」改 AppID；或改 `project.config.json` 的 `"appid"`。
3. **预览 / 真机调试** 确认无误。
4. **上传**（右上「上传」，填版本号 1.0.0 + 备注）。
5. 后台「管理→版本管理」见开发版本 →「选为体验版」可内测。
6. **提交审核**：服务类目建议「教育 / 教育信息服务」或「工具」；功能页 `pages/index/index`；说明「哲学读物，纯本地文字、无网络请求、无信息收集」。
7. 审核通过 → **发布** → 得正式二维码。

## 4. 常见问题
- **AppID 报错**：先用「测试号」，或确认 `project.config.json` 的 `appid` 已填对。
- **想改字号/配色/排版**：正文字号在 `index.wxss` 的 `.bp`（默认 31rpx、行高 1.92）；主色 `#a73e2c`（赭红）与 `#f3ebda`（羊皮纸）。
- **想改文字/补内容**：编辑 `pages/index/data.js` 的 `ENTRIES`（每个条目 `body` 是段落数组，`k:'p'` 为正文、`k:'li'` 为要点）。
- **个人主体**：原生、零网络请求、不用 web-view，不受限制。
- **基础库**：默认 3.5.5；不同版本可在「详情→本地设置→调试基础库」选较新稳定版。
