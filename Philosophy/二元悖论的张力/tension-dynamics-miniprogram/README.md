# 在不可消解的张力中安居 · 微信小程序

十二篇「二元悖论的张力作为系统的动力」动力学分析的原生微信小程序。
界面按手机原生交互设计（自定义导航栏、rpx 适配、明暗双主题、沉浸阅读），**非 web 套壳**。

---

## 一、如何运行（微信开发者工具）

1. 安装并打开「微信开发者工具」（稳定版即可）。
2. 顶部菜单：项目 → 导入项目（或首页「+ 导入」）。
3. **目录**：选择本文件夹（包含 `app.json`、`project.config.json` 的这一层）。
4. **AppID**：当前已填占位值 `touristappid`。
   - 若只是本地预览，直接点「测试号」即可，无需企业/个人 AppID。
   - 若要真机预览或发布，把 `project.config.json` 里的 `appid` 改成你自己的小程序 AppID。
5. 点击「编译」。模拟器会加载首页；点击「预览」可生成二维码用手机微信扫码体验。

> 说明：项目未使用任何 npm 依赖，导入后无需「构建 npm」，开箱即用。

---

## 二、界面设计要点（手机原生）

- **自定义导航栏**：通过 `wx.getMenuButtonBoundingClientRect()` 精确测算胶囊按钮位置，适配刘海屏 / 不同机型；首页导航随滚动由透明渐变为实底。
- **宣纸 / 水墨视觉**：暖白纸色底、朱砂与黛青点缀；三个分类分别用靛蓝（哲学根本）、松绿（科学之眼）、赤陶（存在与现实）色系区分。标题用衬线（宋体），正文用系统无衬线，编辑式排版。
- **明暗双主题**：日间 / 夜读一键切换，偏好持久化（`wx.setStorageSync`）。
- **沉浸阅读页**：
  - 顶部细线显示**阅读进度**；
  - 右下角浮动工具：回到顶部、**字号三档**（小/中/大）、日间夜读、**目录**；
  - **目录抽屉**：底部弹出，列出母悖论、各重张力、元结论，点按平滑跳转（`scroll-into-view`）；
  - 母悖论 / 元结论以印章式徽章高亮，诗化收束居中衬线呈现。

## 三、目录结构

```
app.js / app.json / app.wxss      全局逻辑、配置、设计系统（CSS 变量 + 双主题）
project.config.json / sitemap.json 工程与索引配置
utils/parser.js                    markdown → 结构化区块解析器（原生渲染，非 rich-text）
data/catalog.js                    12 篇元数据 + 3 个分类
data/content.js                    正文索引（id → 正文模块）
data/content/<id>.js               每篇完整正文（module.exports 字符串）
pages/index/                       首页（hero + 分类卡片）
pages/article/                     阅读页（解析渲染 + 进度 + 目录 + 字号/主题）
pages/about/                       框架说明页
```

## 五、如何增删 / 修改内容

- 改正文：直接编辑 `data/content/<id>.js`（内容为 markdown 风格的模板字符串，支持 `#/##/###` 标题、`**加粗**`、`---` 分隔、`- 列表`）。
- 加文章：在 `data/content/` 新增 `<id>.js`，在 `data/content.js` 里 require，在 `data/catalog.js` 的 `articles` 里补一条元数据即可。
