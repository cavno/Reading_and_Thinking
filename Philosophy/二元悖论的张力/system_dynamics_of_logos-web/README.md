# Logos 系统动力学 · 二元张力驱动的哲学系统演化
## 访问地址：https://cavno.github.io/system_dynamics_of_logos/
把哲学（Logos）建成一台系统动力学机器的交互模拟器。单文件 index.html，可传 GitHub Pages（main/root）。

## 系统建模
- **要素**：支点/担保人（四站搬迁）、审计引擎（诘问/充足理由律）、对立应力登记簿（41 组＝张力源/燃料舱）、缝合策略库（S1 特许豁免～S6 矛盾本体化）。
- **约束**：不可能三角 A 普遍必然 / B 彻底可审计 / C 自我完备——弃一角方可运行；图中变灰的角＝当前被弃。
- **边界**：蓝色虚线框随三块界碑（可思→可认识→可说）逐档收缩——校准而非败退。
- **回路**：R1 增强回路（追问→对立→更多追问）；B1 调节回路（张力→缝合→泄压留残差）；红色虚线**自指回路**（审计标准审计自身）＝唯一致命故障，两次闭合＝两次爆点（巴门尼德/休谟），支点闪烁。
- **存量**：确定性 C（两崩、一建制、一有界回升）、张力 T（锯齿＝燃料）、**棘轮 K 只增不减**（三难清单→Γ卷立法→休谟之叉→康德边界声明→哥德尔定理，1931 触顶＝边界定理化）。

## 交互
▶ 演化＝自动播放 18 步事件（前600—1931）；时间轴任意跳转；上一步/下一步；**点图中任一系统件**（支点/审计引擎/登记簿/策略库/三角/三储罐）看机理解剖；底部三条存量曲线随步进移动光标，爆点以红虚线标出。两处「明斯基前夜」（C 与 T 同时高位）在事件文案中点明。
