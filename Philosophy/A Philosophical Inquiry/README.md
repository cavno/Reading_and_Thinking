# 探索生命的意义与价值 · 微信小程序

将哲学文章《探索生命的意义与价值——从虚无中寻找希望与力量》（梅春雷）改编为可部署的微信小程序。

## 项目结构

```
miniprogram/
├── app.js                  全局逻辑（记录系统信息）
├── app.json                全局配置（注册页面、导航栏样式）
├── app.wxss                全局样式与设计变量
├── sitemap.json            索引配置
├── project.config.json     项目配置（⚠️ 需替换 AppID）
├── data/
│   └── content.js          全书内容数据（所有章节集中在此，便于修改）
└── pages/
    ├── index/              首页：脉络中枢（Hero + 核心命题 + 目录）
    │   ├── index.js / .json / .wxml / .wxss
    └── detail/             详情页：数据驱动，按章节 id 渲染不同版式
        └── detail.js / .json / .wxml / .wxss
```

## 交互与脉络设计

- **首页** 是全文脉络的中枢：顶部"虚无之核"图示，中部核心命题（一体两面），下部为目录式六章导航，每章标注"诊断→救治"阶段（DIAGNOSIS / SYMPTOMS / ETIOLOGY / PATHOLOGY / REMEDY / SYNTHESIS）。
- **详情页** 用同一个页面按 `?id=` 参数渲染七种不同版式：
  - 前言：四重根基卡片
  - 第一章：思想家横向时间轴，点击姓名切换其诊断详情（雅各比→尼采→海德格尔…）
  - 第二章：标签切换「五种基本类型 ↔ 十种当代表现」
  - 第三章：四大社会根源 + 韦伯"铁笼"
  - 第四章：致病机制流 + 恶性循环 + 临床数据
  - 第五章：三层出路（社会改造标记为"根本出路"）+ 自我演化四阶段
  - 第六章：意义谱系（命运→理性→信仰→自我→社会）+ 马克思引文收束
- **可回溯**：依赖微信原生返回栈；每章底部有「上一章 / 目录 / 下一章」导航，章节间用 `redirectTo` 切换，返回栈不会无限增长。

## 部署步骤

1. **注册 AppID**：登录 [微信公众平台](https://mp.weixin.qq.com) → 注册"小程序" → 在「开发管理 - 开发设置」中获取 AppID。
2. **安装工具**：下载并安装 [微信开发者工具](https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html)。
3. **导入项目**：打开开发者工具 → 导入项目 → 目录选择本 `miniprogram` 文件夹 → 填入你的 AppID（或先用「测试号」预览）。
4. **替换 AppID**：把 `project.config.json` 里的 `"appid": "touristappid"` 改成你的真实 AppID。
5. **预览 / 真机调试**：工具内点击「预览」生成二维码，手机微信扫码体验。
6. **上传发布**：点击「上传」→ 在微信公众平台「版本管理」提交审核 → 审核通过后发布。

## 说明

- **无网络依赖**：所有内容均为本地数据，无任何网络请求，因此无需在公众平台配置服务器域名，审核更简单。
- **字体**：使用系统字体。iOS 会以宋体（Songti SC）呈现典籍质感；安卓部分机型无衬线字体时会回退为系统默认字体，属正常现象。
- **基础库**：`project.config.json` 中 `libVersion` 设为 `3.0.0`，可在开发者工具「详情 - 本地设置」中按需调整。
- **修改内容**：所有文字集中在 `data/content.js`，改动后无需动页面结构即可生效。
