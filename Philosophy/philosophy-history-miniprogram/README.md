# 西方哲学史考察十讲 · 前四讲微信小程序

依据梅春雷老师 2026 年 3 月至 6 月的四份课程讲义整理，共分析 104 页课件。

## 内容结构

- `首页`：课程总问题、四讲入口、三条跨讲暗线、阅读进度。
- `思想脉络`：以八个动作串起“图腾禁忌 → 宗教改革”的思想演进。
- `四讲精读`：每讲包含核心判断、问题、六个论证段落、要点与思想过渡。
- `概念谱系`：40 个关键概念，支持中文、原文和内容搜索及主题筛选。
- `关于`：讲义来源、整理原则与阅读方法。

## 运行方式

1. 打开微信开发者工具。
2. 选择“导入项目”。
3. 项目目录选择本文件夹。
4. AppID 可使用测试号；项目已配置为 `touristappid`。
5. 点击“编译”即可预览。

## 技术说明

- 原生微信小程序，不依赖 npm、云开发或外部接口。
- 课程内容全部位于 `data/course.js` 与 `data/glossary.js`。
- 阅读进度和收藏通过本机 `wx.setStorageSync` 保存，不上传数据。
- 页面采用纯 WXML/WXSS 绘制，无网络图片依赖。

## 内容编辑说明

本项目是课程阅读版，不替代原始课件。整理时保留讲义的核心判断与思想脉络，合并重复材料，并将人物和概念重组为“核心问题—时代回答—思想张力—下一步过渡”的阅读结构。
