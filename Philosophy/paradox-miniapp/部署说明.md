# 悖论图谱 · 微信小程序

将 https://cavno.github.io/systemtensions/ 的**全部内容**（96 系统 · 842 张力 · 11 分类）重构为原生微信小程序。UI 沿用「四律」系列的手稿风设计语言（羊皮纸 #f3ebda · 赤陶 #a73e2c · 爱琴海蓝 #1d4e5f · 做旧金 #b8893a），零网络、纯本地数据、无 web-view。

## 工程结构

```
paradox-miniapp/
├─ app.js / app.json（tabBar：图谱 + 方法论）/ app.wxss（全局设计变量）
├─ project.config.json（⚠️ appid 为 touristappid，需替换）/ sitemap.json
├─ data/index.js            轻量索引（96 系统的 id/名称/分类/时代/母悖论双极/分包归属）
├─ pages/
│  ├─ home/                 首页：品牌区 + 统计 + 11 个分类卡
│  ├─ category/             分类页：系统列表（≥6 项且多时代时自动按「时代」分组，如王朝史 19 组）
│  └─ about/                方法论页
└─ pkg_a / pkg_b / pkg_c / pkg_d   四个分包（各 <1.4MB），每个含：
   ├─ data.js               该分包全部系统的完整内容（前置/母悖论/张力全文/元结论）
   └─ pages/system + pages/tension
```

**分包分布**：pkg_a 思想与信仰（宗教思想、思想体系、学科反思前半）；pkg_b 学科与演化（学科反思后半、宇宙演化、科技演化）；pkg_c 历史（中国王朝史、历史进程、中国地域）；pkg_d 文艺与经济（影视文学、经济金融、抽象系统）。分包在首次进入对应系统页时由微信自动按需下载，用户无感知。

## 信息架构（针对手机重构，非网页移植）

```
首页(分类总览) → 分类页(系统列表·时代分组) → 系统页 → 张力阅读页
```

- **系统页**：分类章 + 系统名 + 副题 → 前置（默认折叠首段，可展开全文）→ 母悖论双极舞台（青蓝 A ↔ 赤陶 B）+ 论述（默认 4 段，可展开）→ 具体张力卡（双极 + 金色小标题摘要，点击进入全文）→ 元结论（默认 2 段，可展开）。**全部内容都在，折叠只是入口形态。**
- **张力阅读页**：正文 31rpx / 行距 2.02 / 段首缩进 2em 的沉浸阅读；小标题金色加粗；底部「← 上一张力 / 返回系统 / 下一张力 →」，前后翻页用 redirectTo，返回栈不膨胀。5 个无 ↔ 的单极张力（如《黑暗骑士》渡轮实验）自动切换单标题版式。
- 系统页与张力页均支持**转发分享**（onShareAppMessage），落地直达该页。

## 最快看到效果（免注册）

1. 安装微信开发者工具（稳定版）：https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html
2. 解压本 zip → 工具「+ 导入项目」→ 目录选 `paradox-miniapp` → AppID 点「测试号」→ 导入。
3. 模拟器里点分类 → 系统 → 张力即可通读全部内容；右上「预览」扫码上真机。

## 正式上线

1. mp.weixin.qq.com 注册小程序 → 「开发管理→开发设置」取 AppID。
2. 工具「详情→基本信息」填 AppID（或改 `project.config.json` 的 `"appid"`）。
3. 预览/真机调试 → 上传 → 后台「版本管理」设体验版 → 提交审核（类目建议「教育/学习工具」；说明"哲学/历史可视化阅读工具，纯本地数据、无网络请求、不收集任何信息"）→ 发布。个人主体可用。

## 常见问题 / 定制

- **包体**：主包 31KB；分包 994KB / 1344KB / 702KB / 763KB，均远低于 2MB 单包上限。
- **改内容**：全文数据在各 `pkg_*/data.js`（键为系统 id）；首页分类/索引在 `data/index.js`。均为构建产物，格式是 `{name, cat, era, tag, pre[], mp{a,b,d[]}, ts[{n,a,b,sum,ih,body[]}], meta[]}`，段落块 `{t:'h'|'p', x:'文本'}`。
- **字体**：系统衬线栈（iOS 显示宋体质感最佳；安卓部分机型回退默认字体属正常）。
- **控制台 wx:key 提示**：正文段落列表为静态渲染，个别 wx:key 警告不影响功能与审核。
- 基础库默认 3.5.5，无任何 API 依赖新特性。
