# 公理组合矩阵 · 微信小程序 — 部署说明（保姆级）

> 原生小程序，由 PC 版完整移植。把四律当作集合/矩阵，排列组合 → 哲学倾向，打破历史线。
> 文件包：`axiom-combinatorics-miniapp.zip`（解压得文件夹）。

## 0. 工程结构
```
axiom-combinatorics-miniapp/
├─ app.js / app.json / app.wxss      ← 全局入口
├─ project.config.json               ← 工程配置（appid 占位 touristappid）
├─ sitemap.json
└─ pages/index/
   ├─ index.wxml   ← 组合矩阵 + 星图 + 底部抽屉
   ├─ index.wxss   ← 样式
   ├─ index.js     ← 组合逻辑（聚类 / 匹配 / 倾向）
   └─ data.js      ← 101 位哲学家数据（与网页版同源）
```
**移植说明**：PC 版用 SVG 画星图，手机端星图是一块**有界的近正方形**图，正好放进屏幕——所以用绝对定位的 `view` 圆点直接画（无需 Canvas，点按即可选中），其余（4×7 组合矩阵、绿/红/黄＋镶环立场色、随机/浏览全部/重置、底部倾向抽屉）与网页版逻辑一致（已用脚本模拟验证：45 种构型、矩阵 4×8、筛选与聚类结果与网页版相同）。

## 1. 最快看到效果（免注册）
1. 安装**微信开发者工具（稳定版）**：`https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html`
2. 解压本 zip。
3. 打开工具 → 扫码登录 → **「+ 导入项目」** → 目录选解压后的文件夹 → AppID 点 **「测试号」** → 导入。
4. 左侧模拟器即跑：上方点矩阵格子设值、点「任意」放开；下方星图点圆点看倾向；右上「🎲随机 / ⊞全部 / ↺重置」。右上 **「预览」** 可扫码上真机（测试号下仅限本人）。

## 2. 正式上线（与一般小程序相同）
1. **注册账号**：`https://mp.weixin.qq.com` → 立即注册 → 选「小程序」→ 邮箱激活 → 选主体（个人/企业）→ 完成验证 → 在「开发→开发管理→开发设置」复制 **AppID**。
2. **填 AppID**：开发者工具「详情→基本信息」改 AppID；或改 `project.config.json` 里 `"appid": "touristappid"` 为你的 AppID。
3. **真机调试 / 预览** 确认无误。
4. **上传**：工具右上「上传」→ 填版本号（如 1.0.0）+ 备注。
5. 后台 **「管理→版本管理」** 见「开发版本」→ 可「选为体验版」内测。
6. **提交审核**：填服务类目（建议「教育/工具」）、功能页 `pages/index/index`、备注「哲学可视化工具，纯本地数据、无网络请求、无信息收集」。
7. 审核通过 → **「发布」** → 得正式二维码，分享即可。

## 3. 常见问题
- **AppID 报错**：先用「测试号」；或确认 `project.config.json` 的 `appid` 已填对。
- **个人主体**：不能用 `web-view`、部分类目受限——本项目是原生小程序、零网络请求，不受影响。
- **改内容**：立场色在 `index.js` 的 `PIP/pipStyle`；倾向命名在 `index.js` 的 `ISMS`；增删哲学家在 `pages/index/data.js` 的 `C`。
- **基础库**：默认 3.5.5；若工具版本不同，在「详情→本地设置→调试基础库」选较新稳定版（≥2.9.0）。
